What Now?

About

Our project, “What Now?”, is an interactive scheduling assistant that can be used to easily and efficiently create and update the user’s schedule. The project serves as an effective time management tool that will improve one’s productivity and help them stay organized. We will use a variety of Python libraries, such as Kivy, for a user-friendly interface, and CoquiTTS, for text to speech modifications. Other AI libraries will be used for text generation and summarization. Using these different libraries, our project will scan documents for a solid base schedule and take voice or manual input to modify it quickly.

This website will aggregate weekly updates, the documentation, and our GANTT chart for easy access and quick updating.
